# **App Name**: EduVerse

## Core Features:

- Subject Cards: Display subject cards (Mathematics, Science, etc.) on the homepage, arranged in a grid.
- Game Placeholders: Show three placeholder games upon clicking a subject card.
- AI Tutor Placeholder: Implement a UI placeholder for the AI Tutor feature.
- Topic Description Placeholder: Implement a UI placeholder for the Topic Description feature.
- Dashboard Progress: Display progress bars for each subject on the dashboard.
- AI-Powered Tutoring Tool: Enable AI powered tutoring, where the system decides whether to use the AI tutor functionality to augment the app based on student performance.
- Topic Descriptions with Gemini AI: Leverage the Gemini API to generate detailed descriptions for chosen subject matter; The system will use a tool to determine when generating topic descriptions is most valuable for each user.

## Style Guidelines:

- Primary color: Deep violet (#7957D6) to evoke futuristic learning.
- Background color: Dark charcoal gray (#222222) to support a dark theme.
- Accent color: Neon green (#39FF14) to highlight interactive elements.
- Body and headline font: 'Space Grotesk' (sans-serif) for a techy, scientific feel. 
- Use clean, modern icons to represent subjects, achievements, and AI tools.
- Employ a responsive grid layout to adapt across desktop and mobile devices.
- Add subtle transitions and animations to create a smooth, engaging user experience.
